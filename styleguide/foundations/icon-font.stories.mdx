# Icon font

Icons are made available via the `cads-icons` font. To use them apply the appropriate class to the element that is displaying them. You can apply normal CSS styles, e.g. `color` to change their appearance.

## List of icons and classes

Do not forget to prefix the given class name with `cads-icon_`.
