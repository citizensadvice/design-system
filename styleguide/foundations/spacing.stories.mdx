# Spacing & Sizing

## Sizing

All page content should be limited to 1200px wide. Use the `$cads-max-content-size` variable for setting the `width` or apply the `.cads-max-content-width` class to restrict the content of your components on large screen sizes.

## Spacing

Spacing variables can be used directly in your SASS.

| Variable          | Example                                         | Size |
| ----------------- | ----------------------------------------------- | ---- |
| `$cads-spacing-1` | <div class="cads-styleguide-spacing-block-1" /> | 4px  |
| `$cads-spacing-2` | <div class="cads-styleguide-spacing-block-2" /> | 8px  |
| `$cads-spacing-3` | <div class="cads-styleguide-spacing-block-3" /> | 12px |
| `$cads-spacing-4` | <div class="cads-styleguide-spacing-block-4" /> | 16px |
| `$cads-spacing-5` | <div class="cads-styleguide-spacing-block-5" /> | 24px |
| `$cads-spacing-6` | <div class="cads-styleguide-spacing-block-6" /> | 32px |
| `$cads-spacing-7` | <div class="cads-styleguide-spacing-block-7" /> | 40px |
