# Lists

## Unordered lists

Unordered lists should be used to show collections of items where their order does not have meaning, for example the criteria for claiming a particular benefit.

<Preview>
  <Story
    name="Unordered List"
    parameters={{ docs: { source: { code: ulHtml } } }}
  >
    {() => ulHtml}
  </Story>
</Preview>

### Unordered list placeholders

You can use the following placeholders to achieve the correct unordered list styling outside of a `.cads-prose` element.

```scss
@extend %cads-list-unordered; // use for normal unordered lists
@extend %cads-list-unordered-line; // use for inline unordered lists
```

## Ordered lists

Ordered lists should be used to show collections of items where the order does have meaning, for example the steps in a process.

<Preview>
  <Story
    name="Ordered List"
    parameters={{ docs: { source: { code: olHtml } } }}
  >
    {() => olHtml}
  </Story>
</Preview>

### Ordered list placeholders

You can use the following placeholders to achieve the correct ordered list styling outside of a `.cads-prose` element.

```scss
@extend %cads-list-ordered; // use for normal ordered lists
```

## Plain lists (no markers)

In some situations, you will want the styles of an unordered list without the bullet points - for example to present a list of links on a summary page.

<Preview>
  <Story
    name="Plain List"
    parameters={{ docs: { source: { code: noBulletHtml } } }}
  >
    {() => noBulletHtml}
  </Story>
</Preview>

This can be achieved by extending the placeholder below.

```scss
@extend %cads-list-no-bullet;
```
