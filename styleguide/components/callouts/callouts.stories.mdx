## Haml template options

The Haml partial takes a `notice` hash with the following properties:

| Property         | Description                                                       |
| ---------------- | ----------------------------------------------------------------- |
| `notice['type']` | Optional type, one of: `standard`, `example`, `import`, `adviser` |
| `notice['body']` | HTML string                                                       |

For example:

<Source dark="true" lang="haml" code={templateImportant.raw} />
