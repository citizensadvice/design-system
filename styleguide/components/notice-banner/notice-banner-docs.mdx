## Using with Rails

If you are using the `citizens_advice_components` gem you call the component from within a template using:

```rb
= render(CitizensAdviceComponents::NoticeBanner.new(label: "Example label") do
  %p Your HTML content
```

### Component arguments

The component accepts a [`content` block](https://viewcomponent.org/guide/#implementation) for the main contents along with the following arguments:

| Argument | Description            |
| -------- | ---------------------- |
| `label`  | Required, label string |

## Haml template options

The Haml partial takes a `banner` hash with the following properties:

| Property | Description           |
| -------- | --------------------- |
| `label`  | Required, string      |
| `body`   | Required, HTML string |
