## Using with Rails

If you are using the `citizens_advice_components` gem, you can call the component from within a template using:

```rb
= render CitizensAdviceComponents::Sidebar.new(title: "Related Content") do |c|
    c.section_title(title: "Section title", url: "/section-title")
    c.sidebar_links([
      { title: "Link 1", url: "/link-1" },
      { title: "Link 2", url: "/link-2" },
      { title: "Link 3", url: "/link-3" },
      { title: "Link 4", url: "/link-4" },
      { title: "Link 5", url: "/link-5" }
    ])
    "Additional content"
end
```

| Argument               | Description                                            |
| ---------------------- | ------------------------------------------------------ |
| `title`                | Required. Title for the sidebar                        |
| `section_title[title]` | Required. A section title above the links              |
| `section_title[url]`   | Optional. A link for the section title                 |
| `sidebar_links`        | Required, an array of hashes, each with the following: |
| `sidebar_link[url]`    | → Required, url for the sidebar link                   |
| `sidebar_link[title]`  | → Required, title for the sidebar link                 |
